# Student Activity Tracker :

## Project Description
Student Activity Tracker is a front-end web application developed using HTML, CSS, and JavaScript. It allows users to view student activities, mark them as completed, and track progress in real time.

## Features :
- Student-wise Activity Tracking
- Accordion View
- Mark Activity as Completed
- Real-time Progress Update
- Dynamic UI without Page Reload

## Technologies Used :
- HTML5
- CSS3
- JavaScript

## Folder Structure :
Student Activity Tracker
│── index.html
│── style.css
│── script.js
└── README.md

## How to Run :
1. Download the project.
2. Open the project folder.
3. Double-click `index.html` or open it in a browser.
